# PartCCTV, Yet Another CCTV App

## Installation
Фронтэнд развернуть на web сервере

Бэкэнд запустить из PHP-CLI

Зайти в веб-интерфейс и настроить систему

## Requirements
  - PHP 5.6 и выше с расширением ZeroMQ
  - MySQL
